# Inventory Optimization and Decision Making Strategy
**Strategic Inventory Command Center**

### 🎯 Project Overview
This sophisticated 10-tab Supply Chain Analytics Dashboard is designed to optimize inventory health, improve service levels, and mitigate operational risk. It moves beyond static reporting to provide **Integrated Category Intelligence**, leveraging Python to automate complex supply chain decision-making.

### 🛠️ Key Capabilities
* **SOP Engine:** Automated Safety Stock and Reorder Point (ROP) calculations based on target service levels.
* **Risk Mitigation:** Bullwhip Effect detection and ABC-XYZ Value-Volatility classification.
* **Network Optimization:** Multi-echelon stock balancing and warehouse-to-satellite hub analysis.
* **Stress-Testing:** Resilience modeling for geopolitical and lead-time disruptions.

### 📊 Business Impact
* **Capital Efficiency:** Identifying "Dead Stock" to free up working capital.
* **Service Level Assurance:** Balancing holding costs with 95%+ fill rate targets.
* **Data Integrity:** Integrated Audit and Compliance engine for physical-to-system sync.

### 🧪 Tech Stack
* **Language:** Python (Pandas, NumPy, SciPy)
* **Visualization:** Plotly (Interactive Geospatial and Financial Charts)
* **Deployment:** Streamlit Community Cloud
