# Gogo Global Orchestra: Strategic Inventory Command Center

A sophisticated 10-tab Supply Chain Analytics Dashboard built to optimize inventory health and mitigate risk.

### 🎯 Project Overview
This command center moves beyond static reporting to provide **Integrated Category Intelligence**. It leverages Python and Streamlit to automate complex supply chain math, including:
* **Dynamic SOP Engine:** Automated Safety Stock and ROP calculations.
* **Risk Mitigation:** ABC-XYZ Matrix and Bullwhip Effect detection.
* **Resilience Testing:** Stress-testing supply chain scenarios.

### 🛠️ Tech Stack
* **Analytics:** Python (Pandas, NumPy, SciPy)
* **Visualization:** Plotly (Interactive Charts)
* **Deployment:** Streamlit Community Cloud
